## 3. 设计建议

1. 所有主键字段建议统一使用 `bigint`。
2. 需要频繁查询的字段可在后续建表时增加索引，如 `user_id`、`post_id`、`activity_id`、`session_id`。
3. 帖子、评论、举报、活动等涉及状态流转的表，建议统一使用状态字段控制业务逻辑。
